# 系统指令清单

这份文档汇总机器人当前支持的全部群聊指令，并标明每条指令的用途和权限要求。

## 权限说明

- 普通成员：任何群成员都可以使用
- 群管理员：对应群配置里的 `switcherUserIds`
- 超级管理员：`config/groups.json` 顶层 `superAdminUserIds`

说明：

- 群管理员可以使用全部系统指令
- 超级管理员拥有全部权限，并且可以增删群管理员
- 普通成员默认只能用对话、语音、帮助和部分状态查询

## 1. 帮助与总览

### `#功能`

- 权限：普通成员
- 作用：发送系统功能总览

### `#帮助`

- 权限：普通成员
- 作用：等同于 `#功能`

### `#命令`

- 权限：普通成员
- 作用：等同于 `#功能`

### `#功能 列表` / `#帮助 查看` / `#命令 全部`

- 权限：普通成员
- 作用：帮助命令的等价别名
- 说明：这些命令只会返回帮助信息，不会触发节假日或日报主动发送

### `#功能 <分类>` / `#帮助 <分类>` / `#命令 <分类>`

- 权限：普通成员
- 作用：查看某个模块的分类帮助
- 可用分类：`对话`、`语音`、`技能`、`实时对话`、`日报`、`节假日`、`管理员`、`权限`

示例：

```text
#功能 技能
#帮助 管理员
#命令 语音
```

## 2. 对话与语音

### `@机器人 <内容>`

- 权限：普通成员
- 作用：使用当前 skill 正常对话

### `#语音 <内容>`

- 权限：普通成员
- 作用：让机器人先生成回复，再把回复合成为语音发送

### `@机器人 语音说 <内容>`

- 权限：普通成员
- 作用：和 `#语音 <内容>` 等价

## 3. Skill 技能

### `#技能 列表`

- 权限：普通成员
- 作用：查看当前群允许使用的 skill 列表

### `#技能 切换 <skillId>`

- 权限：群管理员 / 超级管理员
- 作用：切换当前群 skill，并清空该群上下文

示例：

```text
#技能 切换 leijun
```

## 4. 实时对话

### `#实时对话 列表`

- 权限：群管理员 / 超级管理员
- 作用：查看当前实时对话用户列表和倒计时

### `#实时对话 添加 <QQ号>`

- 权限：群管理员 / 超级管理员
- 作用：把某个 QQ 加入实时对话跟踪列表

### `#实时对话 移除 <QQ号>`

- 权限：群管理员 / 超级管理员
- 作用：从实时对话列表移除某个 QQ

### `#实时对话 间隔 <分钟>`

- 权限：群管理员 / 超级管理员
- 作用：设置机器人最后一次发言后，需要安静多久才允许自动实时接话

示例：

```text
#实时对话 间隔 5
```

## 5. 群聊日报

### `#日报`

- 权限：普通成员
- 作用：查看日报状态

### `#日报 状态`

- 权限：普通成员
- 作用：查看日报是否开启、何时发送

### `#日报 发送`

- 权限：群管理员 / 超级管理员
- 作用：立即手动发送一次日报

### `#日报 开启`

- 权限：群管理员 / 超级管理员
- 作用：开启群聊日报

### `#日报 关闭`

- 权限：群管理员 / 超级管理员
- 作用：关闭群聊日报

### `#日报 时间 <HH:mm>`

- 权限：群管理员 / 超级管理员
- 作用：设置日报每天自动发送时间

示例：

```text
#日报 时间 18:00
```

## 6. 节假日倒计时

### `#节假日`

- 权限：普通成员
- 作用：直接查看最近 5 个已公布的中国节假日倒计时

### `#节假日 查看`

- 权限：普通成员
- 作用：查看节假日倒计时内容

### `#节假日 状态`

- 权限：普通成员
- 作用：查看是否开启、几点自动发送，并附带当前倒计时内容

### `#节假日 发送`

- 权限：群管理员 / 超级管理员
- 作用：立刻手动发一次节假日倒计时

### `#节假日 开启`

- 权限：群管理员 / 超级管理员
- 作用：开启每天自动发送节假日倒计时

### `#节假日 关闭`

- 权限：群管理员 / 超级管理员
- 作用：关闭每天自动发送节假日倒计时

### `#节假日 时间 <HH:mm>`

- 权限：群管理员 / 超级管理员
- 作用：设置每天自动发送节假日倒计时的时间

示例：

```text
#节假日 时间 09:00
```

## 7. 管理员

### `#管理员`

- 权限：群管理员 / 超级管理员
- 作用：查看本群管理员和超级管理员列表

### `#管理员 列表`

- 权限：群管理员 / 超级管理员
- 作用：查看本群管理员和超级管理员列表

### `#管理员 添加 <QQ号>`

- 权限：超级管理员
- 作用：把某个 QQ 添加为当前群管理员

### `#管理员 移除 <QQ号>`

- 权限：超级管理员
- 作用：把某个 QQ 从当前群管理员里移除

说明：

- 这里的“群管理员”实际存储在 `config/groups.json` 的 `switcherUserIds`
- 超级管理员只能通过配置文件设置，不能通过聊天命令添加

示例：

```text
#管理员 添加 123456789
#管理员 移除 123456789
```

## 8. 补充说明

- 机器人只会在群里被 `@` 时进入正常对话
- 如果同群已有请求在处理，机器人会提示稍后再试，避免乱序
- 语音模式成功时默认只发语音，不再额外补文字
- 节假日倒计时当前只会列出“已公布”的假期安排，不会伪造未来未发布安排
